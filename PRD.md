## 1. 产品概述

### 1.1 产品定义

一个AI驱动的内容自动化创作与分发系统，用户只需输入简单提示词，系统即可自动生成文案、图片、视频，并一键分发到多个国内外社交媒体平台。

### 1.2 核心流程

```
提示词输入
  ↓
AI内容生成（文案+图片+视频）
  ↓
内容预览与编辑
  ↓
多平台自动分发
  ↓
数据追踪与分析
```

### 1.3 目标平台

**国内平台（优先）:**
- 抖音
- B站 (bilibili)
- 微信视频号
- 微信公众号
- 小红书

**国际平台:**
- YouTube
- Twitter/X
- Medium
- Instagram
- TikTok

## 2. 目标用户

### 2.1 主要用户画像

**个人创作者**
- 年龄: 20-40岁
- 职业: 自媒体创作者、博主、UP主
- 痛点:
  - 内容创作效率低
  - 多平台分发繁琐
  - 缺乏专业技能（视频剪辑、平面设计）
  - 预算有限，无法组建团队

**中小企业/工作室**
- 类型: 营销团队、电商、培训机构
- 痛点:
  - 需要大量内容但人力不足
  - 内容质量参差不齐
  - 多平台运营成本高
  - 需要快速响应市场变化

**营销人员**
- 职业: 社交媒体运营、数字营销
- 痛点:
  - 需要持续产出内容
  - A/B测试不同内容形式
  - 数据追踪和优化需求

### 2.2 用户场景

**场景1: 日更内容快速创作**
> 用户是美妆博主，每天需要在小红书、抖音、B站发布内容。使用本系统，早上输入"春季护肤新品推荐"，系统自动生成3篇不同风格的文案、配套图片和15秒视频，并自动发布到三个平台。

**场景2: 产品批量推广**
> 电商公司有100个SKU需要制作推广视频。传统方式需要拍摄剪辑2周。使用本系统，上传产品图片和描述，批量生成100个15秒产品视频，并自动分发到抖音、视频号、小红书。

**场景3: 多语言内容本地化**
> 跨境电商需要将英文产品介绍翻译并本地化为中文、日文、韩文内容。系统使用GLM自动翻译，结合Seedance 2.0生成符合当地文化的视频内容，并发布到相应地区的平台。

## 3. 核心价值主张

### 3.1 效率提升

- **10倍速度:** 从构思到发布，从数小时缩短到数分钟
- **批量处理:** 一次提示词，生成多个版本的内容
- **并行分发:** 一键发布到10+平台

### 3.2 成本节约

- **节省人力:** 减少60-80%的内容制作人力
- **低成本AI:** Seedance 2.0仅需1元/秒，远低于传统制作成本
- **按需付费:** 只为实际使用付费，无固定成本

### 3.3 质量保障

- **AI智能:** 使用业界领先的GLM-4.7、Seedance 2.0
- **多版本生成:** 自动生成多个版本供选择
- **人工审核:** 发布前可预览和编辑

### 3.4 易用性

- **中英双语:** 界面全中文，降低使用门槛
- **零代码:** 无需技术背景，人人可用
- **模板化:** 丰富的预设模板，快速上手

## 4. 用户故事

### 4.1 内容创作

**US-001: 文案生成**
> 作为用户，我希望输入产品名称和特点后，系统自动生成吸引人的文案，这样我就不需要花时间想创意了。

**验收标准:**
- 输入: "iPhone 16 Pro Max，钛金属边框，A18芯片"
- 输出: 3条不同风格的文案（正式、活泼、短小）
- 可编辑和调整
- 支持中英文输出

**US-002: 图片生成**
> 作为用户，我希望根据文案自动匹配或生成高质量图片，这样我就不需要找设计师了。

**验收标准:**
- 自动生成1-3张配图
- 支持自定义风格（写实、卡通、简约等）
- 图片质量1080p以上
- 支持下载和替换

**US-003: 视频生成**
> 作为用户，我希望自动生成15-30秒的短视频，这样我就不需要学习视频剪辑了。

**验收标准:**
- 支持文本生成视频、图片生成视频
- 时长可选：5秒、10秒、15秒、30秒
- 支持多种风格：产品展示、故事叙述、快节奏
- 视频质量720p-1080p
- 生成时间<2分钟

### 4.2 平台分发

**US-004: 单平台发布**
> 作为用户，我希望选择一个平台并自动发布内容，这样我就不需要手动上传了。

**验收标准:**
- 支持选择发布平台
- 自动适配平台格式（尺寸、时长、标题）
- 显示发布状态（成功/失败）
- 发布后返回平台链接

**US-005: 多平台批量发布**
> 作为用户，我希望一次选择多个平台，系统自动同时发布，这样我就不需要一个个操作了。

**验收标准:**
- 支持多选平台（最多10个）
- 并行发布，节省时间
- 每个平台独立状态显示
- 失败自动重试
- 发布进度实时显示

**US-006: 定时发布**
> 作为用户，我希望设置发布时间，系统按时自动发布，这样我就可以提前准备内容了。

**验收标准:**
- 支持选择日期和时间
- 支持时区设置
- 到期前5分钟提醒
- 过期未发布提醒

### 4.3 内容管理

**US-007: 内容库**
> 作为用户，我希望查看所有生成的内容历史，这样我就可以复用或参考了。

**验收标准:**
- 按时间排序显示
- 支持搜索和过滤
- 显示内容详情（文案、图片、视频）
- 支持重新编辑和发布
- 支持删除

**US-008: 模板管理**
> 作为用户，我希望保存常用的提示词为模板，这样我就可以快速重复使用了。

**验收标准:**
- 支持创建自定义模板
- 模板包含提示词、平台选择、风格设置
- 支持模板分类
- 一键应用模板

### 4.4 数据分析

**US-009: 发布追踪**
> 作为用户，我希望查看内容在各平台的表现，这样我就知道哪些内容受欢迎了。

**验收标准:**
- 显示各平台的基本数据（浏览、点赞、评论、转发）
- 数据汇总和对比
- 趋势图表
- 支持导出报告

**US-010: 成本统计**
> 作为用户，我希望查看AI服务的使用成本，这样我就可以控制预算了。

**验收标准:**
- 实时显示当前成本
- 按月统计
- 成本预测（基于历史）
- 成本优化建议

## 6. 非功能需求

### 6.1 性能要求

- **响应时间:**
  - 页面加载 < 2秒
  - 文案生成 < 30秒
  - 图片生成 < 60秒
  - 视频生成 < 120秒
  - 平台发布 < 30秒/平台

- **并发能力:**
  - 支持100个并发用户
  - 支持10个同时生成任务

- **可用性:**
  - 系统可用性 > 99%
  - 计划内维护 < 4小时/月
  - 故障恢复时间 < 1小时

- **数据安全:**
  - 所有API密钥加密存储（AES-256）

  - HTTPS传输
  - 定期安全审计

  - 用户数据隔离

- **访问控制:**
  - 用户身份认证
  - 角色权限管理
  - API访问限流

- **浏览器支持:**
  - Chrome 90+ (推荐)
  - Edge 90+

  - Safari 14+

- **设备支持:**
  - PC/Mac (1920x1080及以上)
  - 平板 (1024x768及以上)
  - 手机 (响应式设计)

### 6.4 可维护性

- **日志系统:**
  - 完整的操作日志
  - 错误日志
  - 性能日志

- **监控系统:**
  - 实时系统状态

  - 异地容灾

## 7. 技术架构

### 7.1 整体架构

```
┌─────────────────────────────────────────┐
│           用户界面层                     │
│  Next.js 15 + next-intl (中英双语)      │
└─────────────────────────────────────────┘
                  ↓ API调用
┌─────────────────────────────────────────┐
│           API网关层                     │
│     Node.js + Express + TypeScript      │
│  - 认证授权                              │
│  - 请求路由                              │
│  - 限流控制                              │
└─────────────────────────────────────────┘
          ↓       ↓       ↓
┌────────────┐ ┌───────┐ ┌──────────┐
│  AI引擎    │ │工作流 │ │分发引擎  │
│            │ │引擎   │ │          │
│ - GLM      │ │- 任务 │ │- 平台   │
│ - DALL-E   │ │  队列 │ │  适配器 │
│ - Seedance │ │- 状态 │ │- 内容   │
│ - HeyGen   │ │  管理 │ │  格式化 │
│ - 可灵     │ │- 错误│ │- 发布   │
└────────────┘ │  处理 │ │  追踪   │
               └───────┘ └──────────┘
                  ↓
┌─────────────────────────────────────────┐
│          数据持久层                      │
│  Prisma ORM + SQLite/PostgreSQL         │
│  - 内容模板                             │
│  - 生成历史                             │
│  - 平台凭证                             │
│  - 任务记录                             │
└─────────────────────────────────────────┘
```

### 7.2 技术栈

**前端:**
- 框架: Next.js 15 (App Router)
- 语言: TypeScript
- 样式: Tailwind CSS
- 组件: shadcn/ui
- 国际化: next-intl
- 状态管理: React Context / Zustand

**后端:**
- 运行时: Node.js 25+
- 框架: Express
- 语言: TypeScript
- ORM: Prisma
- 数据库: SQLite (开发) / PostgreSQL (生产)

**AI服务:**
- 文案: GLM-4.7 (主力) / OpenAI GPT-4 / Gemini Pro
- 图片: DALL-E 3 / Stable Diffusion
- 视频: Seedance 2.0 (主力) / HeyGen / 可灵 / Runway ML

**第三方集成:**
- 平台API: B站、抖音、微信、YouTube、Medium、Twitter
- 自动化: OpenClaw + Chrome DevTools MCP (小红书)
- 存储: 本地 / OSS (阿里云/腾讯云)

### 7.3 部署架构

**开发环境:**
- 本地Docker Compose
- 前端: http://localhost:3000
- 后端: http://localhost:4000

**生产环境 (推荐):**
- 前端: Vercel / Netlify
- 后端: Railway / Render / 阿里云ECS
- 数据库: 云数据库 (PostgreSQL)
- 存储: 云存储 (OSS)

## 8. 平台集成策略

**集成方式:** 官方API
**难度:** ⭐⭐ 中等
**成本:** 免费

**功能:**
- 视频上传
- 投稿提交
- 分区选择
- 标签设置
- 封面上传

**限制:**
- 需要创作者账号
- 需要申请开发者权限
- 有上传大小和时长限制

#### 8.1.2 抖音

**集成方式:** 开放平台API
**难度:** ⭐⭐⭐ 较难
**成本:** 免费（需审核）

**功能:**
- 视频上传
- 发布视频
- 数据统计

**限制:**
- 需要企业资质或个人开发者申请
- 审核严格
- 有发布频率限制

#### 8.1.3 微信视频号

**集成方式:** 开放平台API
**难度:** ⭐⭐⭐ 较难
**成本:** 300元/年（企业认证）

**功能:**
- 视频上传
- 发布视频
- 数据分析

**限制:**
- 需要企业认证
- 个人号无法使用API

## 9. 数据模型

### 9.1 核心实体

#### User (用户)
```
- id: 主键
- email: 邮箱（唯一）
- password: 密码（加密）
- name: 姓名
- avatar: 头像
- language: 默认语言 (zh/en)
- created_at: 创建时间
- updated_at: 更新时间
```

#### ContentTemplate (内容模板)
```
- id: 主键
- user_id: 用户ID（外键）
- name: 模板名称
- description: 描述
- type: 类型 (text/image/video/all)
- prompt_template: 提示词模板
- ai_provider: AI服务选择
- style: 风格设置
- platforms: 平台预设 (JSON)
- created_at: 创建时间
- updated_at: 更新时间
```

#### ContentHistory (生成历史)
```
- id: 主键
- user_id: 用户ID（外键）
- template_id: 模板ID（外键，可为空）
- prompt: 用户输入提示词
- type: 内容类型
- generated_content: 生成内容 (JSON)
  - text: 文案内容
  - images: 图片列表
  - videos: 视频列表
- ai_provider: 使用的AI服务
- cost: 成本（分）
- status: 状态 (draft/generated/published)
- created_at: 创建时间
```

#### PlatformCredential (平台凭证)
```
- id: 主键
- user_id: 用户ID（外键）
- platform: 平台名称
- account_name: 账号名称
- access_token: 访问令牌（加密）
- refresh_token: 刷新令牌（加密）
- expires_at: 过期时间
- status: 状态 (active/inactive/expired)
- created_at: 创建时间
- updated_at: 更新时间
```

#### DistributionRecord (分发记录)
```
- id: 主键
- content_id: 内容ID（外键）
- user_id: 用户ID（外键）
- platform: 平台名称
- platform_post_id: 平台返回的帖子ID
- platform_url: 平台帖子URL
- status: 状态 (pending/success/failed)
- error_message: 错误信息
- published_at: 发布时间
- created_at: 创建时间
```

#### WorkflowTask (工作流任务)
```
- id: 主键
- user_id: 用户ID（外键）
- workflow_type: 工作流类型
- input_data: 输入数据 (JSON)
- status: 状态 (pending/running/completed/failed)
- current_step: 当前步骤
- result: 结果 (JSON)
- error: 错误信息
- created_at: 创建时间
- completed_at: 完成时间
```

#### ScheduledPost (定时发布)
```
- id: 主键
- user_id: 用户ID（外键）
- content_id: 内容ID（外键）
- platforms: 平台列表 (JSON)
- scheduled_time: 计划发布时间
- timezone: 时区
- status: 状态 (pending/sent/failed/cancelled)
- created_at: 创建时间
```

### 9.2 数据库关系图

```
User (1) ----< (N) ContentTemplate
User (1) ----< (N) ContentHistory
User (1) ----< (N) PlatformCredential
User (1) ----< (N) WorkflowTask
User (1) ----< (N) ScheduledPost

ContentTemplate (1) ----< (N) ContentHistory
ContentHistory (1) ----< (N) DistributionRecord
ContentHistory (1) ----< (N) ScheduledPost
```

## 11. 成本预算

### 11.1 开发成本

**人力成本:**
- 全栈开发 x 1人: 150-200小时
- UI设计: 20小时
- 测试: 30小时
- **总计: 200-250小时**

**时间成本:**
- MVP开发: 4-6周
- 完整功能: 8-12周

### 11.2 运营成本 (月度)

**必需成本:**
- VPS主机: ¥35-70元 ($5-10)
- 域名: ¥7-15元 ($1-2)
- **小计: ¥42-85元**

**AI服务成本 (按使用量):**
- 文案生成 (GLM): ¥70-350元 ($10-50)
- 图片生成 (DALL-E): ¥35-140元 ($5-20)
- 视频生成 (Seedance 2.0): ¥150-450元 ($20-70)
  - 30个5秒视频: ~¥150元
- **小计: ¥255-940元**

**平台成本:**
- 微信企业认证: ¥300元/年
- 其他平台: 免费
- **小计: ¥25元/月**

**总运营成本:**
- **最低:** ¥322元/月 (~$45)
- **平均:** ¥756元/月 (~$105)
- **最高:** ¥1,610元/月 (~$225)

### 11.3 成本优化策略

1. **智能AI选择:**
   - 文案优先用GLM（已有，性价比高）
   - 视频优先用Seedance 2.0（1元/秒）
   - 批量测试用可灵免费版

2. **缓存机制:**
   - 相似提示词复用结果
   - 减少30-40%重复生成

3. **批量生成:**
   - 一次性生成多天内容
   - 降低单次成本

4. **预算控制:**
   - 设置月度上限
   - 超标自动暂停
